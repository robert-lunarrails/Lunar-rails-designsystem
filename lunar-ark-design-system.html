10 — AI Rules

Do's and Don'ts for AI

Rules the AI must follow when building any Lunar Ark product, page, or component.

✅ DO
  • Use ALL CAPS for all headings, hero text, and navigation links
  • Use --gradient-brand (Pink → Violet) on hero text and CTAs
  • Use --color-bg-page (#000000) — always pure black
  • Use Barlow Condensed for ALL headings and nav, Barlow Semi Condensed for body
  • Use sharp corners — --radius-sm to --radius-lg (2–8px)
  • Apply --shadow-glow-violet on card and button hover
  • Reference Bitcoin confidently — it's the core product
  • Use the arc/orbit motif from the logo as a decorative element
  • Use wide letter-spacing (--tracking-widest) on hero text
❌ DON'T
  • Use white or light backgrounds — dark-theme-only brand
  • Use Inter, Roboto, Arial, Satoshi — Barlow family only
  • Use lowercase or sentence case for headings — ALL CAPS is the standard
  • Apply gradient to body text or anything below --text-h3
  • Use excessive border-radius — max is --radius-2xl (20px)
  • Hard-code hex values — always use CSS tokens
  • Mix pink and violet randomly — gradient always Pink → Violet
  • Use glassmorphism as primary pattern — nav backdrop-blur only
  • Add drop shadows to text
  • Use more than 2 accent colors in a single section
