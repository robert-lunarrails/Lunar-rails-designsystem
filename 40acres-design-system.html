02 — Design Philosophy

Light Mode · Gradient on Buttons Only

40Acres is a clean editorial brand. The site is white and grey. Photography provides visual impact in heroes. The gradient appears in exactly one place.

✅ Gradient is used here
--gradient-btn: linear-gradient(180deg, #6C2DB7 0%, #4D73D8 100%)
Direction: 180deg (vertical, top→bottom) · border-radius: 0
✓ Primary CTA buttons
✓ Form submit buttons
✓ Nav "Become a Client" button
❌ Gradient does NOT appear here
✕ Hero headings — white text on photography
✕ Section headings on light pages
✕ Section backgrounds or overlays
✕ Decorative dividers or accents
✕ Card borders, icon fills
✕ Atmospheric washes
Primary Button — Key Details
✅ linear-gradient(180deg, #6C2DB7 0%, #4D73D8 100%)
✅ border-radius: 0 — sharp corners, brand standard
✅ Vertical direction — top violet, bottom blue
❌ Not border-radius: 8px
❌ Not horizontal gradient

09 — AI Rules

Do's and Don'ts for AI

The most important rules for building any 40Acres product.

✅ DO
  • Treat this as a light-mode brand — white and grey backgrounds
  • Use full-bleed photography or video in hero sections
  • Place plain white headings over photos — no gradient fill
  • Use the gradient only on CTA buttons — vertical direction, 0px radius
  • Use #342E51 for all headings on white/grey sections
  • Use → arrow links for service items and navigation
  • Alternate sections: white → grey → white → grey
  • Dark background only in the footer
❌ DON'T
  • Apply gradient to hero text, any heading, or body copy
  • Use gradient as a background, overlay, or decoration
  • Round CTA button corners — border-radius must be 0px
  • Make the site dark-mode except the footer
  • Add neon glow, purple atmospheric effects, or dark overlays
  • Use gradient more than once per screen view
  • Use a horizontal gradient on buttons — it must be vertical (180deg)
  • Hard-code hex values — always use CSS tokens
